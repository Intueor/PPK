Для корректной сборки и обслуживания проекта на Linux требуется отредактировать параметры проекта.
Сборка:
    + Этап сборки:
	Команда:		    /usr/bin/bash
	Параметры:		    %{sourceDir}/deployment %{ActiveProject:Name} %{Env:HOME} %{ActiveProject:BuildConfig:Path} %{CurrentDocument:Path}
    + Этап очистки:
	Команда:		    /usr/bin/bash
	Параметры:		    %{sourceDir}/cleanup %{ActiveProject:Name} %{Env:HOME} %{CurrentDocument:Path}
Запуск:
    Рабочий каталог:		    %{Env:HOME}/.%{ActiveProject:Name}/

При сборке проект полностью инсталлируется в каталог текущего пользователя в скрытую одноимённую проекту папку. При очистке - аналогичная деинсталляция.

Сборка документации с помощью doxygen - по файлу Doxyfile.
Можно добавить в редакторе в [Инструменты\Внешние\Настроить...\Добавить] doxygen с параметрами [%{CurrentDocument:Path}/Doxyfile], затем горяч. клавишу.

При свежей сборке в Doxyfile и *.desktop все уникальные для пользователя пути вписываются вместо трафаретов.
Перед отправкой коммита обязательна очистка проекта - все пути в соотв. файлах будут заменены на трафареты обратно.
